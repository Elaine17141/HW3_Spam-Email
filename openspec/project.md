# Project
Spam Email Classifier

## Tech Stack

- Node.js
- Jest for testing
- ESLint for linting
- Prettier for formatting

## Conventions

- Code should be formatted with Prettier.
- All code should pass ESLint checks.
- All new features should be accompanied by tests.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run tests:
   ```bash
   npm test
   ```

3. Lint files:
   ```bash
   npm run lint
   ```

4. Format files:
   ```bash
   npm run format
   ```
